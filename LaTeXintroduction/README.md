##Make a pdf file

1. Go the Files directory

2. Type: 

*pdflatex labMeetingMikMay2015.pdf*

Done

If you don't see citation, then 

3. Type:

 *bibtex labMeetingMikMay2015.aux*

4. Retype:

 *pdflatex labMeetingMikMay2015.pdf*
